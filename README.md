# EfficientPyTorch

## Install
```bash
git clone https://github.com/hustzxd/EfficientPyTorch.git

cd EfficientPyTorch
# ./setup.sh
source setup.sh
pip install -r requirements.txt
```

## Run

```bash
./run_cli.sh *.prototxt
```
